Справочник "Технологические операции"

Справочник содержит перечень всех производственных технологических операций, выполняемых на предприятии, и предназначен для целей посменного планирования производства и учета оплаты сдельного труда.

Для обеспечения посменного планирования производства следует заполнить следующие реквизиты: «базовая единица измерения» и «норма времени». Здесь норма времени исполнения операции является справочным значением и используется для заполнения времени исполнения этапа в маршрутной карте значением по умолчанию.

Табличных частей справочник не имеет, обязательных реквизитов не содержит. Последовательность заполнения – произвольная.
